# Digital Wellness

A frontend project that promotes healthier digital habits through a clean and simple interface.

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` - Main entry page
- `styles.css` - App styling
- `src/` - Source code (components/pages)

## Run Locally

1. Install dependencies:
```bash
npm install
```
2. Start development server:
```bash
npm run dev
```
